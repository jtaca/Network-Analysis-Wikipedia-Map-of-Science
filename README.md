# Map of Science Network: Analysing the similarity beween Wikipedia science pages.

## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install networkx, matplotlib and numpy.

```bash
pip install networkx
pip install matplotlib
pip install numpy
```

## Data

The data for this experiment was based on the one from 
Calderone, Alberto (2020): A Wikipedia Based Map of Science. figshare. Dataset. https://doi.org/10.6084/m9.figshare.11638932.v5 

Extracted from:
https://figshare.com/articles/A_Wikipedia_Based_Map_of_Science/11638932


The values currespond to data from 18th January 2020.
